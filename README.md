# Json-Shopingcart

An interactive shopping cart web application built using **HTML, CSS, and JavaScript**.

## Features
- Add items to cart
- Remove items from cart
- Clear cart
- Automatic subtotal calculation
- 5% discount if bill exceeds ₹10,000
- Persistent cart using localStorage

## Technologies Used
- HTML
- CSS
- JavaScript
- JSON
- localStorage
